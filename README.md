## Overview

> A **stunning**, **fully interactive** clone of the Apple iPhone 15 Pro website featuring **smooth animations**, **immersive models** and a **sleek**, **responsive** design that mirrors the **premium Apple experience**.

## Features

- Beautiful Smooth Animations
- 3D Model Rendering
- Custom Video Carousel
- Modern UI/UX
- Completely Responsive
- Cross-Device Compatibility

## Quick Start

#### 1 - Clone repo

```bash
git clone <repo_url>
```

#### 2 - Install dependencies

```bash
npm install
```

#### 3 - Run server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the project.
